# Реализация дизайна - Hero секция с фото и красным прямоугольником

## Описание дизайна

На основе вашего описания создана hero-секция с:

1. **Фоновое изображение** - фотография человека
2. **Текст поверх изображения** - заголовок и описание
3. **Красный прямоугольник** - акцентный элемент
4. **CTA кнопка** - кнопка призыва к действию

---

## Что создано

### Компонент `HeroSection.astro`

**Структура:**
- Фоновое изображение с overlay (затемнение для читаемости текста)
- Текст поверх изображения (заголовок + описание)
- Красный прямоугольник как акцентный элемент
- CTA кнопка внизу

**Особенности:**
- Адаптивный дизайн
- Градиент overlay для лучшей читаемости текста
- Тень на тексте для контраста
- Красный прямоугольник (80x80px, можно настроить размер и цвет)

---

## Настройка

### 1. Добавить фоновое изображение

Сохраните фотографию в `public/images/hero-bg.jpg` или укажите свой путь:

```astro
<HeroSection 
  imageUrl="/images/your-photo.jpg"
  ...
/>
```

### 2. Настроить красный прямоугольник

В `HeroSection.astro` можно изменить:
- Цвет: `background-color: #ff4444;` (строка 100)
- Размер: `width: 80px; height: 80px;` (строка 98-99)
- Позицию: через flexbox в `.hero-content`

### 3. Настроить текст

Измените props компонента:
```astro
<HeroSection 
  title="Ваш заголовок"
  description="Ваше описание"
  ...
/>
```

## Параметры для настройки

### Цвет красного прямоугольника:
Текущий: `#ff4444`
Можно изменить на:
- `#ff0000` - чистый красный
- `#dc2626` - более темный красный
- `#ef4444` - красный Tailwind
- Любой другой оттенок

### Размеры:
Текущий: `80x80px`
Можно изменить на:
- `60x60px` - меньше
- `100x100px` - больше
- Или другие размеры

### Позиционирование:
Текущий: справа от текста (flexbox)
Можно изменить через CSS
