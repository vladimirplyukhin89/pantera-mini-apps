# Pantera — фронтенд (Astro)

Статическая сборка (`output: 'static'`), React-острова, контент из Strapi.

```bash
npm install
npm run dev    # http://localhost:4321
npm run build
```

Переменные: `astro-app/.env` (`STRAPI_URL`, `STRAPI_TOKEN`). Обзор проекта, деплой и CI — в [../docs/README.md](../docs/README.md).
