# Тестирование Telegram WebApp SDK

## Важно понимать

**Telegram WebApp SDK работает ТОЛЬКО внутри Telegram Mini App!**

В обычном браузере:
- Скрипт загрузится ✅
- Но объект `window.Telegram.WebApp` может быть недоступен ❌
- Функционал не будет работать ❌

---

## Как проверить что SDK подключен

### 1. Проверка в браузере (частичная)

1. Запустите dev сервер:
   ```bash
   npm run dev
   ```

2. Откройте в браузере: `http://localhost:4321`

3. Откройте консоль разработчика (F12 → Console)

4. Проверьте в консоли:
   ```javascript
   window.Telegram
   ```

**Результат:**
- В браузере: `undefined` или объект без WebApp (это нормально!)
- В Telegram Mini App: объект с методом `WebApp`

### 2. Проверка через код (добавлен компонент TelegramWebAppInit)

Я создал компонент `TelegramWebAppInit.astro` который:
- Проверяет наличие `window.Telegram.WebApp`
- Логирует информацию в консоль
- Инициализирует WebApp если он доступен

**В консоли вы увидите:**
```
✅ Telegram WebApp SDK загружен
WebApp: [object Object]
Версия: 6.0
Платформа: ios/android/web
```

Или (в браузере):
```
⚠️ Telegram WebApp SDK не найден (это нормально в браузере)
SDK работает только внутри Telegram Mini App
```

## Реальное тестирование в Telegram

### Для полного тестирования нужно:

1. **Развернуть приложение** (например, на Vercel, Netlify, или другом хостинге)

2. **Создать Telegram Bot** через @BotFather

3. **Настроить Mini App** в BotFather:
   - Команда: `/newapp`
   - Выбрать бота
   - Указать название и описание
   - Указать URL приложения (ваш хостинг)
   - Получить ссылку типа: `https://t.me/your_bot/your_app`

4. **Открыть Mini App в Telegram**

5. **Проверить в консоли** (через веб-инспектор Telegram)

---

## Что можно проверить сейчас (в браузере)

### 1. Скрипт загружается

Проверьте в DevTools → Network:
- Должен быть запрос к `https://telegram.org/js/telegram-web-app.js`
- Статус: 200 OK

### 2. CSS переменные Telegram (если используются)

В браузере переменные не будут работать, но в Telegram Mini App они автоматически установятся:
```css
background-color: var(--tg-theme-bg-color, #ffffff);
```

Fallback значение `#ffffff` будет использоваться в браузере.

### 3. Структура HTML правильная

Проверьте что скрипт в `<head>`:
```html
<script src="https://telegram.org/js/telegram-web-app.js"></script>
```

## Полезные команды для проверки в консоли

```javascript
// Проверить что SDK загружен
window.Telegram

// Проверить WebApp объект
window.Telegram?.WebApp

// Информация о пользователе (в Telegram)
window.Telegram?.WebApp?.initData

// Версия SDK
window.Telegram?.WebApp?.version

// Платформа
window.Telegram?.WebApp?.platform

// Получить данные пользователя (в Telegram)
window.Telegram?.WebApp?.initDataUnsafe?.user
```
